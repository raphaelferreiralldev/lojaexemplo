# NOVA — Loja Online de Demonstração

Loja online de exemplo construída com Next.js, React, TypeScript e Tailwind CSS.

**[Ver loja ao vivo](https://app-six-mu-67.vercel.app/)**

## Funcionalidades

- Vitrine de produtos com filtro por categoria
- Página individual de cada produto
- Carrinho de compras persistente (localStorage)
- Checkout com cartão de crédito, Pix e boleto

## Tecnologias

- [Next.js 16](https://nextjs.org) — framework full-stack
- [React 19](https://react.dev) — interface
- [TypeScript](https://www.typescriptlang.org) — tipagem
- [Tailwind CSS](https://tailwindcss.com) — estilização
- [Vercel](https://vercel.com) — hospedagem gratuita

## Rodar localmente

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).
